# Get the head of an fst file set

Get the head of an fst file set

## Usage

``` r
head_fst(path, n = 5, i = 1)
```

## Arguments

- path:

  The directory to search.

- n:

  Number of rows to show.

- i:

  Which output output chunk to show.
