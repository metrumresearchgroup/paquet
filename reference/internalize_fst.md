# Get the contents of an fst file set

Get the contents of an fst file set

## Usage

``` r
internalize_fst(path, ...)

get_fst(path, ...)
```

## Arguments

- path:

  The directory to search.

- ...:

  Other arguments passed to
  [`fst::read_fst()`](http://www.fstpackage.org/reference/write_fst.md).
