# Configure a locker directory

Configure a locker directory

## Usage

``` r
config_locker(where, ask = NULL, noreset = NULL)
```

## Arguments

- where:

  The locker location.

- ask:

  Logical; toggles whether or not the user will be asked to confirm
  prior to resetting a locker space.

- noreset:

  Logical; toggles whether or not a locker space is resettable.
